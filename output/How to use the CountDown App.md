# How to use the CountDown App

### In this article, we'll explain how to use the CountDown app on OptiSigns.

* [What You'll Need](#WhatYouNeed)
* [Creating a CountDown Asset](#Creating)
  + [Theme Settings](#ThemeSettings)
  + [Advanced Settings](#Advanced)

The CountDown (or Count Down) app is a simple way to display a customizable count down clock for any occasion.

![OptiSigns count down timer example](https://support.optisigns.com/hc/article_attachments/45147769373075)

The clock can display any title you like, and when it goes off, can display any message you like. Here, we'll show how to create one. Let's get started.

---

## What You'll Need

* An [**OptiSigns-capable device**](https://support.optisigns.com/hc/en-us/articles/360021855653-What-hardware-and-devices-are-supported)
* An OptiSigns [**Standard plan or higher**](https://www.optisigns.com/pricing)
* A screen, [**set up and paired**](https://support.optisigns.com/hc/en-us/articles/18823504383891-OptiSigns-Getting-Started-Guide)

## Creating a CountDown Asset

From the OptiSigns portal, go to **Files/Assets** **→ Apps:**

![optisigns files and assets](https://support.optisigns.com/hc/article_attachments/45147769375379)

On the Apps menu, select **CountDown:**

![Countdown app location in optisigns](https://support.optisigns.com/hc/article_attachments/45147743659795)

The below screen should open up:

![countdown app appearance in optisigns](https://support.optisigns.com/hc/article_attachments/45147743661203)

These are the options, and what they do. Any edits made to these options should be automatically reflected in the **Preview** pane on the right. The **Preview** can be displayed in either **Landscape** or **Portrait** format.

* **Name:** The name of the CountDown asset. This is for saving the asset in OptiSigns, and the text entered here will not appear on the asset itself.
* **Text:** The text to be displayed on the CountDown asset while it counts down. By default, this says "OptiSigns Timer," and it is in this area this message will be displayed.
* **End Time:** When selected, will bring up a calendar:

  ![calendar end time](https://support.optisigns.com/hc/article_attachments/45147743663379)

  This allows you to choose the precise date and time for the countdown to be completed.
* **Message:** The message to be displayed at the end of the countdown.
* **Timer Style:** Lets you choose the type of Timer to display. Choose between **Flip**, **Basic**, **Ring**, and **Slide:**

  \
  ![timer styles count down app](https://support.optisigns.com/hc/article_attachments/45147743666323)\

  Specific differences between these can be seen in the Preview panel.
* **Days Only:** Check this box to only show how many days are left on your countdown timer.
* **Theme:** Lets you change between a number of basic themes. Scroll down for more. Their specifics can be seen in the Preview panel.

### **Theme Settings**

\
![theme settings count down app](https://support.optisigns.com/hc/article_attachments/45147743672851)

* **Primary Color:** Determines the background color. Can be chosen with Hex Code or via color picker:

  \
  ![colour picker countdown app](https://support.optisigns.com/hc/article_attachments/45147769390611)
* **Secondary Color:** Determines the text color. Can be chosen with Hex Code or via color picker.
* **Font Size:** Choose between Default Font Size, or Custom. When Custom is selected, will provide a new option: **Custom Font Size**.

  \
  **![font size countdown app](https://support.optisigns.com/hc/article_attachments/45147743678099)**

  + **Custom Font Size:** Choose your font size.
* **Transparency:** Slider which allows the transparency of the image to be changed.

### **Advanced Settings**

Expandable panel. Click this for even more options:

\
![advanced settings countdown app](https://support.optisigns.com/hc/article_attachments/45147769396883)

* **Recurring:** When checked, enables recurring events to be repeated at selected intervals. Opens up more options:

  \
  ![recurring box countdown app](https://support.optisigns.com/hc/article_attachments/45147769398163)

  + **Repeat Every:** Lets you choose how often the countdown repeats. You can adjust this number, and choose from **Day, Week, Month,** or **Year**. When **Week, Month,** or **Year** is chosen, it will show what days it will **Repeat On**.

    \
    \
    **Week:**\
    ![repeat on weekly](https://support.optisigns.com/hc/article_attachments/45147743684371)\
    **Month:**\
    ![repeat on monthly](https://support.optisigns.com/hc/article_attachments/45147769400211)\
    **Year:**\
    **![repeat on yearly](https://support.optisigns.com/hc/article_attachments/45147743686419)**
  + **Ends:** Lets you choose when a recurring countdown timer will end. This can be **Never**, **On**, or **After**.
    - **Never:** The countdown timer will repeat as per above instructions and will not stop recurring until asset is changed.
    - **On:** Allows you to set a specific date at which time the countdown timer will stop recurring.
    - **After:** Allows you to set a specific number of recurrences after which the timer will stop recurring.

      |  |
      | --- |
      | **NOTE** |
      | When setting a recurring element, the Message element will display until 12:00 AM the next day before the countdown timer starts again. |
* **Show Date:** When checked, will show the date on the countdown timer. Also provides **Date Format** option.

  \
  **![show date format countdown app](https://support.optisigns.com/hc/article_attachments/45147743688083)**

  + **Date Format:** Allows you to choose a format for the displayed date.
* **Background Image:** Allows you to choose between a **Default** Background Image and a **Custom** one. Selecting **Custom** will enable the **Choose Photo** option.\
  ![choose photo optisigns countdown app](https://support.optisigns.com/hc/article_attachments/45147743690643)
  + **Choose Photo:** Lets you use an existing OptiSigns image asset as a Background Image for your countdown timer. This will override any other Background Image options, including Background Color, etc.
* **Language:** Choose the language to display clock details such as **Hour**, **Minute**, and **Date** from a variety of language options. These include English, Deutsch(German) ,Español(Spanish), Français(French), Português(Portuguese), Svenska(Swedish), Italiano(Italian), Nederlands(Dutch), and 中文(Chinese)

After you've configured the CountDown app to your liking, hit **Save**. Your CountDown app will appear as an Asset in OptiSigns, and it can now be directly assigned to a screen or used in a Playlist. This asset can then be edited at any time from the Files/Assets tab.
